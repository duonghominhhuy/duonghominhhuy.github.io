- Array Extensions

- UIApplication Extensions
- UIButton Extensions
- UIColor Extensions
- UIImage Extensions
- UILabel Extensions
- UIScrollView Extensions
- UISearchView Extensions
- UIStackView Extensions
- UITableView Extensions
- UITextView Extensions
- UIView Extensions
- UIViewController Extensions
